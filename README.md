# ⌨️ Typing Master — Soft Pastel Focus Edition

A beautifully styled, distraction-free, and fully functional Typing Master application designed to test and accelerate your words-per-minute (WPM) speed and accuracy. This web app features a cool, minimal glassmorphic workspace tailored for fluid typing experiences across all digital screens.

---

## ✨ Features

- **Cool & Soft Theme:** Built with a premium Nordic slate, lavender, and mint pastel color palette designed to prevent eye strain during long typing sessions.
- **Real-Time Analytics Grid:** Tracks live operational metrics including continuous countdown timer updates, exact dynamic WPM, and precise character accuracy ratios.
- **Fluid Micro-Interactions:** Smooth CSS transitions and an elegant pulsating active cursor pinpointing exactly where your layout stroke focus rests.
- **Responsive Mobile Docking:** Engineered with an adaptive structural grid that scales natively from high-end wide screens down to small mobile viewports.
- **Cross-Device Input:** Uses an intelligent hidden input matrix allowing seamless execution on both desktop physical keys and mobile virtual layouts.
